# Vanta Void

A Roblox script hub for combat enhancement, ESP visuals, and miscellaneous utilities.

## Status

**BETA** — Expect bugs, crashes, and incomplete features. Active development is ongoing. Report issues to help improve the script.

## Supported Executors

| Executor | Status |
|----------|--------|
| Potassium | Tested |
| Volt | Confirmed Working |
| Wave | Confirmed Working |
| Volcano | Confirmed Working |

Other executors may work but are not verified.

## Features

- **Combat**
  - Aimbot with smoothness and prediction
  - FOV circle with snaplines (only draws when target is inside FOV)
  - Lock-on target mode
  - Customizable hitbox selection (Head, Body, Random)

- **Visuals**
  - ESP with boxes, tracers, skeletons
  - Player name, health, distance, and weapon display
  - Configurable max distance

- **Misc**
  - Full bright / no fog
  - NoClip
  - WalkSpeed modifier
  - Self charms (first-person arm/gun coloring)

- **UI**
  - Draggable interface with ghost outline
  - Tab-based navigation
  - Key system with save option
  - Discord integration
